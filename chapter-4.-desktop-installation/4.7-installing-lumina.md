# 4.7 Installing Lumina

## Installation

```
# pkg install lumina xorg lightdm lightdm-gtk-greeter wqy-fonts 
```

## configure

```
# sysrc dbus_enable="YES"
# sysrc lightdm_enable="YES"
```

```
# ee ~/.xinitrc
```

Add.

```
exec lumina-desktop
```
