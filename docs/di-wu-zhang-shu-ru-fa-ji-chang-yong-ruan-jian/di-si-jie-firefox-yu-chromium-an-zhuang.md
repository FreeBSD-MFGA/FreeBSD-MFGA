# 第四节 Firefox 与 Chromium 安装

## 火狐浏览器

### 安装普通版本：
```
# pkg install firefox
```

或者

```
# cd /usr/ports/www/firefox
# make install clean 
```

### 安装长期支持版本：

```
pkg install firefox-esr
```
或者

```
#cd /usr/ports/www/firefox-esr/ && make install clean
```

## Chromium

```
# pkg install chromium
```
或者

```
# cd /usr/ports/www/chromium
# make install clean
```
