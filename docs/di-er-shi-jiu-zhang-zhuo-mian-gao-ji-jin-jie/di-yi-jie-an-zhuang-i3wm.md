# 第一节 安装 i3wm

## 安装 i3wm

```
# pkg install -y xorg i3
```

或者通过 ports 安装：

```
# cd /usr/ports/x11-wm/i3/ 
# make install clean
```

## 配置

```
$ echo "/usr/local/bin/i3" >> /usr/home/你的用户名/.xinitrc
$ chown 你的用户名 /usr/home/你的用户名/.xinitrc
```

## 启动

可以用 `startx` 启动 i3 了。
